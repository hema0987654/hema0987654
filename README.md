### 📐 Engineering Mindset

- **System Design First** – I design the architecture before writing code, considering data flow, separation of concerns, and future scalability.
- **Clean, Testable Code** – Every module is built with maintainability in mind, following SOLID principles and modular patterns.
- **Performance Conscious** – Database query optimization, efficient caching strategies, and mindful resource usage are part of my daily workflow.
- **Production Ownership** – From CI/CD pipelines to monitoring, I treat my APIs as products that need to be reliable, observable, and secure.
